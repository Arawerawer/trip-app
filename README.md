# 旅遊行程規劃 App

一個輕量、直覺的旅遊行程規劃工具，支援離線使用的 Progressive Web App (PWA)。

## ✨ 功能特色

- 📅 **多天行程規劃** - 輕鬆管理多日旅遊計畫
- 🕒 **時間與地點記錄** - 為每個行程設定時間和地點
- 🗺️ **Google Maps 整合** - 一鍵查看地點位置
- 💾 **自動儲存** - 所有資料自動保存到本地，不怕遺失
- 📱 **離線使用** - 安裝為 PWA，無網路也能使用
- 🎨 **流暢動畫** - 精心設計的互動體驗
- 📱 **響應式設計** - 支援手機、平板、桌面裝置

## 🚀 快速開始

### 線上使用

直接訪問網站，即可立即開始使用。首次使用建議將應用安裝到主畫面：

**在手機上（iOS/Android）：**

1. 使用瀏覽器開啟網站
2. 點擊「加到主畫面」或「安裝」
3. 從主畫面啟動，就像原生 App 一樣使用

**在電腦上（Chrome）：**

1. 點擊網址列右側的安裝圖示
2. 確認安裝
3. 應用會以獨立視窗開啟

### 本地開發

# 安裝依賴

npm install

# 啟動開發伺服器

npm run dev

# 構建生產版本

npm run build

# 預覽構建結果（測試 PWA）

npm run preview

## 📖 使用指南

### 第一次使用

1. **選擇旅行開始日期**

   - 在首頁點擊日期選擇器
   - 選擇你的旅行開始日期

2. **新增第一個行程**

   - 輸入行程名稱
   - 選擇時間
   - 輸入地點
   - 點擊「確定」

3. **開始規劃**
   - 點擊「開始規劃行程」進入規劃頁面

### 管理行程

#### 新增行程

1. 找到想要新增行程的日期卡片
2. 點擊「新增行程」按鈕
3. 填寫行程資訊後點擊「確定」

#### 編輯行程

1. 點擊行程卡片右上角的筆型圖示
2. 修改資訊後點擊「確定」

#### 刪除行程

1. 點擊行程卡片右上角的垃圾桶圖示
2. 行程會播放動畫後刪除

#### 查看地點

- 點擊行程卡片中的地圖圖示，會在新分頁開啟 Google Maps

### 清除所有資料

1. 點擊右上角的垃圾桶圖示
2. 確認清除
3. 所有行程和日期都會被刪除，返回首頁

## 🛠️ 技術架構

### 核心技術

- **Vue 3** - 使用最新的組合式 API
- **TypeScript** - 完整的類型安全
- **Vite** - 極速的開發體驗
- **Pinia** - 直覺的狀態管理
- **Vue Router 4** - 現代化路由
- **Tailwind CSS** - 實用優先的樣式設計

### PWA 功能

- Service Worker 自動生成
- 離線優先快取策略
- 支援安裝到主畫面
- 自動更新機制

### 資料持久化

使用 `pinia-plugin-persistedstate` 自動將所有行程資料保存到瀏覽器的 `localStorage`，確保：

- 關閉瀏覽器後資料不會遺失
- 離線狀態下也能讀取資料
- 跨裝置同步（如果使用同一瀏覽器帳號）

## 💡 使用提示

1. **資料安全**：所有資料只保存在你的裝置上，不會上傳到任何伺服器
2. **瀏覽器支援**：建議使用 Chrome、Edge、Safari 或 Firefox 最新版本
3. **清除資料**：如果遇到問題，可以使用「清除所有資料」重新開始
4. **離線使用**：安裝為 PWA 後，即使沒有網路也能正常使用（除了 Google Maps 功能）
